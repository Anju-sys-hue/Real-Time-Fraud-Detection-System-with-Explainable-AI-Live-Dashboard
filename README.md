# Real-Time Fraud Detection System with Explainable AI & Live Dashboard

## Project Overview

This project presents an AI-powered fraud detection system designed for real-time transaction monitoring and explainable fraud analytics.

The system combines:
- Machine Learning fraud prediction
- Risk segmentation
- SHAP Explainable AI
- Interactive Streamlit dashboard

---

## Features

### Fraud Detection Models
- LightGBM
- XGBoost
- Isolation Forest

### Risk Segmentation
Transactions categorized into:
- Critical Risk
- Suspicious
- Clear

### Dashboard Features
- Executive Overview
- Fraud KPIs
- Transaction Explorer
- Live Risk Scoring
- SHAP Explainability
- Interactive Plotly Visualizations
